# 🚀 Responsive Landing Page

A modern and fully responsive landing page developed using HTML5, CSS3, and JavaScript. This project was created as part of the CodSoft Web Development Internship (Level 1 - Task 2) to demonstrate clean UI design, responsive layouts, and fundamental front-end development skills.

## ✨ Features

- Responsive design for Mobile, Tablet, and Desktop
- Modern navigation bar
- Hero section with call-to-action buttons
- About section with professional profile
- Services section
- Skills section with progress bars
- Experience timeline
- Portfolio/Works section
- Testimonials section
- Contact form
- Resume preview popup
- Smooth scrolling
- Interactive hover effects
- Clean and well-structured code

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📱 Responsive Design

The website is optimized for different screen sizes and provides a smooth user experience across desktops, tablets, and smartphones.

## 🎯 Project Objective

The objective of this project was to build a visually appealing and user-friendly landing page while applying HTML, CSS, and JavaScript concepts such as responsive layouts, navigation, styling, animations, and interactive components.
